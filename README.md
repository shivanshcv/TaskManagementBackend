<h2>Task Management Backend</h2>
This backend application is developed using Java SpringBoot 

<h4>Features</h4>

<ul>
<li>Sign up or Create an account</li>
<li>Login</li>
<li>Create a task</li>
<li>View all tasks</li>
<li>Update a task</li>
<li>Delete a task</li>
<li>Assign a status to each task</li>
</ul>

<h4>Tools Used</h4>
<ol>
<li>Spring Boot</li>
<li>PostgreSQL</li>
<li>Spring Data JPA</li>
<li>Hibernate</li>
<li>JUnit</li>
<li>Postman</li>
<li>Git</li>
</ol>
